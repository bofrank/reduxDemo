# reduxDemo

A demonstration project showcasing Redux state management patterns and best practices.

## Features

- Clear examples of Redux actions, reducers, and selectors
- Integration with React components
- Store configuration and middleware setup
- Best practices for state management

## Installation

```bash
git clone https://github.com/bofrank/reduxDemo.git
cd reduxDemo
npm install
```

## Getting Started

```bash
npm start
```

Then open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

```
src/
├── actions/      # Redux action creators
├── reducers/     # Redux reducers
├── components/   # React components
└── store.js      # Redux store configuration
```
